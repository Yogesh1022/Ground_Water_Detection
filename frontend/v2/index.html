Core Capabilities

Built for Water Security

From GPS coordinates to actionable insights — a fully autonomous prediction pipeline powered by spatial AI and deep learning.

GPS-Based Real-Time Predictions

Enter any GPS coordinate — the system finds 5 nearest wells via KNN, interpolates with IDW, fetches live weather, and runs parallel ensemble inference in <400ms.

10-Model Ensemble

XGBoost + LSTM + CNN-LSTM + GRU + 1D-CNN — weighted ensemble achieves R² = 0.92, beating any single model.

60–90 Day Early Warning

Predict groundwater depletion 3 months ahead using Vidarbha's hidden monsoon-to-recharge lag pattern.

Interactive Crisis Maps

650+ wells color-coded by risk level on live maps — Safe 🟢, Warning 🟠, Critical 🔴, Extreme 🟣.

Lag Feature Engineering — The Core Innovation

Vidarbha's basalt geology creates a hidden 2-3 month delay between monsoon rainfall and groundwater recharge. Our models explicitly capture this with engineered lag features.

SHAP Explainability

Understand exactly why each prediction was made. Full feature importance ranking with SHAP values.
